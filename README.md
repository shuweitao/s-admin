## 项目运行 pnpm方式(推荐)
1. 全局安装pnpm 
npm i pnpm -g
2. 安装依赖
pnpm install 
3. 运行项目
pnpm dev
4. 打包项目
pnpm build
5. 本地预览 打包项目
pnpm preview
